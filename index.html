Loading Slidify

To load, install packages of slidify and create new project

install.packages("devtools")
library(devtools)
install_github('slidify','ramnathv')
install_github('slidifyLibraries','ramnathv')

author("project name")

After the slides have been done, use the below command:

slidify("index.Rmd")
browseURL("index.html")

Errors

If you encounter problem like the below:

#output file: index.md

#Error in if (body$content == "") { : 
#  missing value where TRUE/FALSE needed

Use the fix below:

install_github('ramnathv/slidify', ref = github_pull(425))
